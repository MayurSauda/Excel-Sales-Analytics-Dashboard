# Excel-Sales-Analytics-Dashboard
Interactive Sales Analytics Dashboard built using Excel, Power Query, Pivot Tables, KPI Cards, Timelines, and Slicers.
📊 Sales Analytics Dashboard
Project Overview
This project is an interactive Sales Analytics Dashboard developed in Microsoft Excel.

The dashboard helps analyze:

Monthly Sales Trends
Department Performance
Regional Sales Distribution
Product Category Performance
KPI Metrics

The project demonstrates data cleaning, transformation, analysis, and dashboard development skills using Excel.
